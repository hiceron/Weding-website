# Wedding Planner Web App Tasks

This document outlines the tasks required to build the wedding planner web application, broken down into phases.

## Phase 1: Project Setup & Core UI

*   [ ] **Task 1.1:** Initialize Next.js project with TypeScript. (US001)
*   [ ] **Task 1.2:** Set up Tailwind CSS and Shadcn/ui. (US007)
*   [ ] **Task 1.3:** Create the main layout and navigation components. (US007)
*   [ ] **Task 1.4:** Implement the theme toggle for light and dark modes. (US006)
*   [ ] **Task 1.5:** Set up Firebase project and connect it to the app (Auth, Firestore, Functions, Hosting).

## Phase 2: Authentication & Dashboard

*   [ ] **Task 2.1:** Implement user authentication with Firebase Authentication. (US001)
*   [ ] **Task 2.2:** Create the dashboard page with a countdown timer. (US002)
*   [ ] **Task 2.3:** Add the to-do list progress card to the dashboard. (US002)
*   [ ] **Task 2.4:** Add the guest list summary card to the dashboard. (US002)

## Phase 3: To-Do List Feature

*   [ ] **Task 3.1:** Create Firebase Functions for CRUD operations on tasks. (US003)
*   [ ] **Task 3.2:** Build the to-do list page UI. (US003)
*   [ ] **Task 3.3:** Implement the "Add Task" functionality using Firestore. (US003)
*   [ ] **Task 3.4:** Implement task completion, priority, and deletion using Firestore. (US003)
*   [ ] **Task 3.5:** Build the task detail modal with notes and sub-tasks. (US008)
*   [ ] **Task 3.6:** Implement adding, editing, and deleting sub-tasks in Firestore. (US008)
*   [ ] **Task 3.7:** Add task recommendations feature. (US009)

## Phase 4: Guest List & Invitations

*   [ ] **Task 4.1:** Create Firebase Functions for CRUD operations on guests. (US004)
*   [ ] **Task 4.2:** Build the guest list page UI. (US004)
*   [ ] **Task 4.3:** Implement the "Add Guest" functionality using Firestore. (US004)
*   [ ] **Task 4.4:** Implement guest editing and deletion using Firestore. (US004)
*   [ ] **Task 4.5:** Design the wedding invitation template. (US005)
*   [ ] **Task 4.6:** Implement the invitation sender functionality (e.g., using a "mailto" link or a Firebase Function with an email service). (US005)

## Phase 5: Finalization & Deployment

*   [ ] **Task 5.1:** Ensure the application is fully responsive. (US010)
*   [ ] **Task 5.2:** Conduct thorough testing of all features.
*   [ ] **Task 5.3:** Deploy the application to Firebase Hosting.
